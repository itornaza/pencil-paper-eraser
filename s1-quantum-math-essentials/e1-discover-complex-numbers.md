# s1.e1 - Discover Complex Numbers

## Quadratic equation
```math
\begin{align}
αx^2+βx+γ=0 , \quad Δ=β^2-4αγ \tag{1}
\end{align}
```

### i) $Δ=0$:
```math
x = \frac{-β}{2α} \tag{2}
```

### ii) $Δ>0$:
```math
x=\frac{-β\pm\sqrt{Δ}}{2α} \tag{3}
```

### iii) $Δ<0$:
```math
x =\frac{-β\pm i\sqrt{|Δ|}}{2α} \tag{4}
```

```math
Re(x)=\frac{-β}{2α} \tag{5}
```

```math
Im(x)=\frac{\sqrt{|Δ|}}{2α} \tag{6}
```

## Imaginary numbers

```math
i=\sqrt{-1} \tag{7}
```

